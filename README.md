# PasswordManager_No.7
幫你管理、保存密碼，避免他人無意間看到、直接開啟

## 環境需求
java8以上

## 下載
目前沒有

## 功能
- 以個別檔案加密保存密碼
- 可自訂產生密碼
- 閒置過久自動關閉
- 複製密碼過久自動清除
